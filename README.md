# MyBookShelf
책벌레와 함께하는 독서기록장

1. 상세 설명
    
    네이버 책 검색 API를 통해 도서를 검색해 해당 도서의 독후감을 적을 수 있다.
    
    firebase realtimeDB를 이용해 DB에 독서 기록을 저장, 수정, 삭제 할 수 있다. 
    
    DB에 저장된 독후감의 갯수에 따라 책벌레를 성장시킬 수 있다.
    
    ( 알 → 금간 알 → 누에고치 → 아가벌레 → 어린이벌레 → 성인벌레 )
    
    앱을 킬 때마다 책벌레 밑에 독서 명언을 랜덤으로 보여준다.
